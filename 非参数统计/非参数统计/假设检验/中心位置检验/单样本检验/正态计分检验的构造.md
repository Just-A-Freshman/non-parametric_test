# 正态计分检验的构造
## 1.回顾符号秩检验
- 统计量：符号秩检验采用的统计量本质是$W = |W^+ - W^-|$。例如，在双边检验时，拒绝域是$W > A$。由于$W^+ + W^- = B$，通过分类讨论可以推知拒绝域等价于：
$W^- < C$或$W^+ < C \implies min(W^-, W^+) < C$

- 大小表示：符号秩检验采用正秩表示数据的绝对大小，原有正负符号作为其方向。例如，$[-12, -34, 10, 58]$，先排序得到正秩：
$$|10| < |-12| < |-34| < |58|$$使用数据原有方向后，最终映射: $$-12 \rightarrow -2, \quad -34 \rightarrow -3, \quad 10 \rightarrow 1, \quad 58 \rightarrow 4$$

## 2.正态计分检验
正态计分检验继承了符号秩检验的统计量设计，但在大小表示上，它不直接使用正秩表示数据的**绝对大小**，而是先将正秩转化成分位数，然后再映射到正态分布的分位数点上。但有以下两个关于分位数$M$计算的问题需要注意：

- 分位数的上界。我们知道$\Phi^{-1}(1) = + \infty$。对于正秩最大的数，如上述例子中的$58$，如果$M = \frac{4}{4} = 1$，那么它正态分位数点$\Phi(M) = +\infty$，这显然不合理。为使$M \neq 1$，一个简单的操作是给分母加1。于是，$58$的分位数就变成了$\frac{4}{4+1} = 0.8$
- 分位数下界。因为数据的绝对大小必须使用正数表示。而当$M \leq 0.5$时，$\Phi^{-1}(M) \leq 0$。为了让正秩都能映射到一个正的分位数点，必须令$M > 0.5$

综上，对于$M$的计算，我们需要将正秩：$\{1, 2, 3, ...n\}$，映射到$(0.5, 1)$这个区间内。记正秩为$r$，$r \in [1, n]$。容易构造：
$$M(r) = \frac{1}{2}(1 + \frac{r}{n+1})$$于是，数据的绝对大小映射到了$\Phi^{-1}(M(r))$。记原始数据为$X_i$，$H_0: M_e = \theta$，令$Y_i = X_i - \theta$，则统计量为：
$$NS^+ = \sum_{i=1}^n Sign(Y_i) \cdot \Phi^{-1}(M_i)$$

这实际上就是$|W+ - W^-|$去掉绝对值后的稍加变形，它被称为"正态计分函数"。记$S_i = Sign(Y_i) \cdot \Phi^{-1}(M(r_i))$.

正如符号秩检验的期望和方差完全由$n$决定。由于$\Phi^{-1}(M_i)$完全取决于秩，$NS^+$的期望和方差也完全由$n$决定。仿照符号秩检验，我们可以将$NS^+$写作：
$$NS^+ = \sum_{r=1}^n a_r \cdot s_r$$其中:
- $a_r = \Phi^{-1}(\frac{1}{2} (1 + \frac{r}{n+1}))$
- $s_r = Sign(Y_{D_r})$

在$H_0$成立的条件下，我们有$s_r$之间相互独立，且$P(s_r = -1) = P(s_r = 1) = \frac{1}{2}$，于是可以得到：
$$E(s_r) = \frac{1}{2}(-1 + 1) = 0, Var(s_r) = \frac{(-1)^2 + (1)^2}{2} = 1$$将$a_r$视为常数，$s_r$视为随机变量，可以得到：
- 期望
$$E(NS^+) = E(\sum_{r=1}^n a_r \cdot s_r) = \sum_{r=1}^n a_r \cdot E(s_r) = 0$$

- 方差
$$
Var(NS^+) = Var(\sum_{r=1}^n a_r \cdot s_r) = \sum_{r=1}^n a_r^2 Var(s_r) = \sum_{r=1}^n a_r^2
$$

很多人对正态计分检验有一个误解，认为$NS^+$的方差是"**通过样本估计的**"。那为何我们不用无偏方差：$S^2 = \frac{1}{n-1} (X_i - \bar{X})^2$呢？这正是因为，此时的样本方差就是理论方差，根本不存在"估计"一说。

$NS^+$的精确分布是复杂的。但直觉上由于进行了正态分数的压缩，它在形状上与正态接近，并且具有渐近正态性。一般情况下，我们可以近似认为：
$$NS^+ \sim N(0, \sum_{r=1}^n a_r^2)$$

## 4. 对比总结
正态计分检验是一种"半参数"检验，它的效果部分依赖于分布假定。相比其他常见检验：
- ①正态计分$NS^+$有更好的大样本性质
- ②当分布接近正态分布时，检验效果略强于$W^+$检验；接近均匀分布时，效果显著强于符号秩检验；
- ③其他分布条件下，效果不如$W^+$，但略强于纯粹的参数检验，如$t$检验。稳健性由于其"半参数"性质居中。
