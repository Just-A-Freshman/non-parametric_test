# K-S检验
## 1. 检验问题
$H_0: 样本来自所给分布\leftrightarrow H_1：样本不来自所给分布$

## 2. 检验统计量
$$D = \max(|F_n(x) - F(x)|)$$

显然，$D$越大，越应该拒绝原假设。$D$统计量服从$D$分布，这是一个人为构造的分布，需要查表得到临界值。

## 3. 性质
### 3.1 大样本近似
当$n$足够大时，$D$分布的临界值可以用如下公式近似：
$$D_{\alpha, n} = 1.36 / \sqrt{n}$$

### 3.2 稳健性
$K-S$检验既不依赖于均值的位置，对尺度化也不敏感，适用范围广；

### 3.3 与$\chi^2$检验的比较
- $\chi^2$检验主要用于类别数据，非类别数据需要人为进行分组；$K-S$检验主要用于连续型数据
- 由于$K-S$检验无需分组，直接对$n$个样本进行检验。其有效性通常要大于$\chi^2$检验。
